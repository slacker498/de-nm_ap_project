MATLAB Simulation & Code

Understanding the numerical modeling of RLC circuits in MATLAB using standard second-order ODE solvers.

1. System Reduction & Solver Logic

MATLAB solvers (like `ode45`) require second-order differential equations to be reduced to a system of first-order ordinary differential equations. To accomplish this, we define state variables representing the charge \(q(t)\) and the current \(i(t)\).

Let the state vector be:

$$\mathbf{y} = \begin{bmatrix} y_1 \\ y_2 \end{bmatrix} = \begin{bmatrix} q(t) \\ \frac{dq}{dt} \end{bmatrix}$$

Differentiating \(\mathbf{y}\) yields the first-order system:

$$\frac{d\mathbf{y}}{dt} = \begin{bmatrix} \frac{dy_1}{dt} \\ \frac{dy_2}{dt} \end{bmatrix} = \begin{bmatrix} y_2 \\ \frac{F(t) - R y_2 - \frac{1}{C} y_1}{L} \end{bmatrix}$$

This system of first-order differential equations can be implemented directly inside a MATLAB function. The solver `ode45` (which uses a variable-step Runge-Kutta-Fehlberg 4th/5th-order formula) then integrates this system over a time span to approximate the solution.

2. Complete Simulation Script

Here is the clean, complete MATLAB script which simulates all the damping cases described in the report. This code uses parameters \(L = 1\text{ H}\), \(C = 0.01\text{ F}\), and solves for underdamped, critically damped, and overdamped conditions. 

% ==========================================================
% ASHESI UNIVERSITY - APPLIED PROGRAMMING FOR ENGINEERS
% RLC SERIES CIRCUIT SIMULATION AND SOLVER COMPARISON
% ==========================================================

clc; clear; close all;

% --- Circuit Parameters ---
L = 1;          % Inductance (H)
C = 0.01;       % Capacitance (F)
tspan = [0 10]; % Time range for integration (seconds)
y0 = [0; 0];    % Initial conditions: q(0) = 0, q'(0) = 0

% ==========================================================
% CASE 1: Underdamped Circuit with AC Voltage Source
% R = 2 ohms (Underdamped since R < 20 ohms)
% Voltage Source: V(t) = 100 * sin(60 * t) V
% ==========================================================
R_case1 = 2;
ode_ac = @(t, y) [y(2); (100*sin(60*t) - R_case1*y(2) - (1/C)*y(1))/L];
[t_ac, y_ac] = ode45(ode_ac, tspan, y0);

% ==========================================================
% CASE 2: Underdamped Circuit with DC Voltage Source
% R = 2 ohms, Voltage Source: V = 100 V
% ==========================================================
R_case2 = 2;
ode_dc_under = @(t, y) [y(2); (100 - R_case2*y(2) - (1/C)*y(1))/L];
[t_dc_under, y_dc_under] = ode45(ode_dc_under, tspan, y0);

% ==========================================================
% CASE 3: Critically Damped Circuit with DC Voltage Source
% R = 20 ohms, Voltage Source: V = 100 V
% ==========================================================
R_case3 = 20;
ode_dc_critical = @(t, y) [y(2); (100 - R_case3*y(2) - (1/C)*y(1))/L];
[t_dc_critical, y_dc_critical] = ode45(ode_dc_critical, tspan, y0);

% ==========================================================
% CASE 4: Overdamped Circuit with DC Voltage Source
% R = 100 ohms, Voltage Source: V = 100 V
% ==========================================================
R_case4 = 100;
ode_dc_over = @(t, y) [y(2); (100 - R_case4*y(2) - (1/C)*y(1))/L];
[t_dc_over, y_dc_over] = ode45(ode_dc_over, tspan, y0);

% --- Plotting Results (similar to Appendix II) ---
figure;
subplot(2,2,1);
plot(t_ac, y_ac(:,1), 'b-', 'LineWidth', 1.5);
grid on; title('Case 1: Underdamped AC (R = 2 \Omega, V = 100sin(60t))');
xlabel('Time t (s)'); ylabel('Charge q(t)');

subplot(2,2,2);
plot(t_dc_under, y_dc_under(:,1), 'c-', 'LineWidth', 1.5);
grid on; title('Case 2: Underdamped DC (R = 2 \Omega, V = 100V)');
xlabel('Time t (s)'); ylabel('Charge q(t)');

subplot(2,2,3);
plot(t_dc_critical, y_dc_critical(:,1), 'g-', 'LineWidth', 1.5);
grid on; title('Case 3: Critically Damped DC (R = 20 \Omega, V = 100V)');
xlabel('Time t (s)'); ylabel('Charge q(t)');

subplot(2,2,4);
plot(t_dc_over, y_dc_over(:,1), 'r-', 'LineWidth', 1.5);
grid on; title('Case 4: Overdamped DC (R = 100 \Omega, V = 100V)');
xlabel('Time t (s)'); ylabel('Charge q(t)');
