# RLC Circuit Dynamics Website

This project is a static website for exploring RLC circuit behavior, including the interactive visualizer and explanatory pages.

## Run locally

1. Open the project folder in your terminal.
2. Start a simple local web server from the project root:

   ```bash
   python3 -m http.server 8000
   ```

3. Open your browser and visit:

   ```text
   http://localhost:8000/
   ```

## Files

- index.html: homepage
- model.html: model and derivation
- code.html: MATLAB code
- results.html: results and analysis
- team.html: team information

## Notes

Because the site uses local assets and browser-based scripts, serving it from a local web server is the easiest way to view it correctly.
